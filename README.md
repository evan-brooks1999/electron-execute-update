## FAQ

**Is Electron safe to use?**  
Electron is distributed as-is for educational and testing purposes. You are responsible for following Roblox's Terms of Service. The software is not described as containing malicious code, but you use it at your own risk.

**Will Electron work with the latest Roblox update?**  
The auto-update engine is intended to keep compatibility aligned with the latest Roblox builds. If something stops working, look for a newer release in the repository.

**How does Electron compare to other free executors?**  
Electron uses no key system, includes a built-in hub with 500+ scripts, and adds persistent queueing plus batch execution. Those functions are not always present in other free tools.

**Can my Roblox account be banned?**  
Third-party execution tools go against Roblox's Terms of Service and can lead to account suspension or termination. Use Electron carefully and consider alternate accounts.

**Where are my scripts stored?**  
User-added scripts are kept locally in the `scripts/` folder. The script hub index is cached under `hub/` and can be refreshed manually.
